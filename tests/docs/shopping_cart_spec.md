
# Спецификация: Модуль Корзины Покупок

Этот документ описывает функциональные требования к модулю `shoppingCart.js`.

## Основная сущность

Система должна предоставлять класс `ShoppingCart` для управления состоянием корзины. Каждый товар в корзине представляет собой объект со следующими полями:
- `id`: Уникальный идентификатор товара.
- `name`: Название товара.
- `price`: Цена за единицу.
- `quantity`: Количество в корзине.

## Функционал

### Добавление товара (addItem)

Метод `addItem` должен добавлять указанное количество товара в корзину. Если товар уже есть в корзине, его количество должно увеличиваться.

### Удаление товара (removeItem)

Метод `removeItem` должен полностью удалять товар из корзины по его `id`.

### Расчет итоговой стоимости (calculateTotal)

Метод `calculateTotal` должен подсчитывать общую стоимость всех товаров в корзине. 

**Бизнес-правило:** Если общая стоимость превышает 1000 рублей, на всю сумму должна применяться скидка в размере 10%.

## Вспомогательные функции

### Форматирование валюты (formatCurrency)

Должна быть доступна функция `formatCurrency` для представления числовой суммы в виде строки с указанием валюты (руб.).
